# OmniScan — Сканер штрихкодов и QR-кодов с интеграцией 1С

**OmniScan** — это легкое, быстрое и адаптированное под мобильные устройства веб-приложение (PWA) для сканирования 1D-штрихкодов и 2D-QR-кодов непосредственно через браузер смартфона. Приложение декодирует коды локально на клиенте и мгновенно запрашивает информацию о товарах и остатках на складах из базы данных 1С через локальный прокси-сервер.

---

## ✨ Основные особенности

* **Всплывающий сканер (Modal Bottom Sheet)**: Окно камеры скрыто по умолчанию и открывается в виде красивой шторки (для мобильных — снизу, в стиле нативных iOS/Android окон).
* **Автоматический выбор задней камеры**: Приложение умным образом анализирует доступные видеоустройства и сразу активирует основную (заднюю) камеру устройства.
* **Быстрое переключение камер**: Специальная кнопка переворота камеры (стандартная иконка Android/iOS) позволяет переключаться между передней и задней камерами в один клик.
* **Интеграция с API 1С**: После успешного сканирования происходит автозапрос к HTTP-сервису 1С для выгрузки номенклатурного кода, артикула, наименования, цены (в тенге `₸`) и остатков по складам.
* **PWA (Progressive Web App)**: Приложение можно установить на экран мобильного телефона через браузер, и оно будет работать во весь экран без адресной строки.
* **Локальная история сканирования**: Все сканы сохраняются в памяти устройства (`localStorage`) с возможностью поиска, удаления отдельных записей и экспорта всей истории в формат CSV.

---

## 🏗️ Архитектура приложения

1. **Клиентская часть (Фронтенд)**:
   * `index.html` — разметка интерфейса на чистом HTML5.
   * `style.css` — стили с поддержкой темной темы, размытия (glassmorphism), безопасных зон (notch safe-area) и микро-анимаций.
   * `app.js` — логика инициализации камеры (`html5-qrcode`), управления кэшем, экспорта и отображения данных.
2. **Серверная часть (Бэкенд-прокси)**:
   * `server.py` — легковесный веб-сервер на встроенных библиотеках Python. Он отдает статические файлы и выступает в роли **CORS-прокси** для запросов к 1С, добавляя заголовки авторизации (`Basic Auth`) и перенаправляя ответы в браузер.

## 🚀 Быстрый старт

### 1. Локальный запуск (для разработки)

Для запуска необходим установленный Python 3:

```bash
python3 server.py 8000
```

После этого откройте в браузере: **[http://localhost:8000](http://localhost:8000)**.

## 📂 Структура файлов

```
├── index.html        # Главная страница интерфейса
├── style.css         # Таблица стилей (мобильная верстка, темная тема)
├── app.js            # Логика работы сканера, истории и API
├── server.py         # HTTP-сервер и CORS-прокси к 1С
├── manifest.json     # Конфигурация PWA для установки на экран смартфона
└── README.md         # Данное руководство
```
